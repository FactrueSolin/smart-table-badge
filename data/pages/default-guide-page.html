默认页面 · 无业务页面时自动展示

HTMLPush 大屏 / 手机 HTML 展示系统

当前还没有可展示的业务页面，所以系统先显示这张默认说明页。 你可以从管理后台上传 HTML、在线编辑页面、切换当前展示内容，适合做活动页、宣传页、设备屏展示与临时页面推送。

开始使用

  1. 点击左上角管理后台进入后台。
  2. 上传一个 HTML 文件，或在编辑器中直接创建新页面。
  3. 在页面列表中切换当前展示页，前台会实时刷新。
  4. 需要生成页面时，可在后台使用 AI 生成功能辅助制作。

项目说明

  • 实时展示基于当前页配置进行前台渲染，适合展示终端持续播放。
  • HTML 驱动核心内容是标准 HTML 文件，便于快速迁移与复用。
  • 后台可维护支持页面上传、切换、编辑，以及 AI 辅助生成。
入口：/admin
能力：上传 / 编辑 / 切换 / AI 生成
状态：暂无业务页面时自动启用默认说明页
